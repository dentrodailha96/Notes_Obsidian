### How to make comparisons 

#### Vergleiche
» Positiv (Grundform):
Heute ist es ==so/genauso/nicht so== warm ==wie== gestern.

| expression | translate |
| ---------- | --------- |
| so         | as        |
| genauso    | as        |
| nicht so   | not as    |
| fast so    | almost as |
| doppelt so | twice     |
| halb so    | half as   |

» Komparativ 
Mario läuft schneller ==als== Peter. 

als = than 

» Superlativ 

Rudi läuft ==am== schnell==sten==.

München ist ==die== schöns==te== Stadt.
